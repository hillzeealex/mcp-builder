# dev-utils

A small toolbox of offline developer utilities, exposed over MCP.

> Generated by mcp-builder. Edit the handler stubs in `src/` to implement your logic.

## Capabilities

| Kind     | Name            | Description                                                  |
| -------- | --------------- | ------------------------------------------------------------ |
| tool     | `hash_text`     | Compute a cryptographic hash of a string.                    |
| tool     | `generate_uuid` | Generate one or more random UUID v4 strings.                 |
| tool     | `format_json`   | Pretty-print or minify a JSON string.                        |
| resource | `about`         | Metadata describing the dev-utils server.                    |
| prompt   | `code_review`   | Ask Claude to review a snippet of code for bugs and clarity. |

## Getting started

```bash
npm install
npm run build
npm start
```

Implement your logic by filling in the `TODO` handlers under `src/tools`, `src/resources`, and `src/prompts`.

## Inspect locally

The [MCP Inspector](https://github.com/modelcontextprotocol/inspector) is the fastest way to exercise your server:

```bash
npm run inspect
```

## Connect to Claude Desktop

Build the server, then add it to `claude_desktop_config.json` (use an **absolute** path):

```json
{
  "mcpServers": {
    "dev-utils": {
      "command": "node",
      "args": ["/ABSOLUTE/PATH/TO/dev-utils/dist/index.js"]
    }
  }
}
```

Restart Claude Desktop. The server's tools, resources, and prompts will appear in the client.
